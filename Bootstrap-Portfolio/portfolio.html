    Portfolio

    To visit the application websites, click on the project icons. Each project also has a short desciption and a link to their Github repositories.

    Crystal Collector Game

    The Crystals Collector Game utilizes a combination of Javascript and jQuery code. A random number is given to the user at the start of each round, and the user must use the four crystals to add their total to equal the random number chosen. Of course, the four crystals will randomize every round, so keep track of the number behind each crystal. Visit the Github repository here.

    Word Guess Game

    The Word Guess Game is based around automotive terminology and utilizes vanilla Javascript. The application will give a hint to the user and the user must use the keyboard to fill in the blanks for the chosen word. Visit the Github repository here.

    Astronomy Trivia Game

    This application utilizes a combination of Javascript, jQuery, and an API. The user must answer a series of astronomy/physics based questions in a limited amount of time. Scores are displayed at the end, and the user may replay the game if they want. Visit the Github repository here.

    GifTastic

    GifTastic was built around Javascript, jQuery, and API calls. This application appends new buttons next to the predefined buttons and allows the user to click and generate 10 Gif images at a time. The application talks to an API, which returns the appropriate search results as searched. The Gif images are initially still, but once clicked, they become animated. Visit the Github repository here.

    Train Scheduler

    Train Scheduler utilizes the Moment.js library along with a Firebase database to store data on travel information. The application will take in several parameters and store them in the database. It will also calculate the next train arrival. Then it captures a snapshot from the database and displays the data back to the user. Visit the Github repository here.
